Key Features Demonstrated:
  • Search Functionality: Search by ASN, SCN, or Container Number with multiple values
  • Multi-Value Input: Enter multiple search values separated by comma or newline
  • Parsed Values Display: Search values shown as removable chips
  • Search Options: Exact match and case-sensitive options
  • Results Preview: Shows count of found documents before applying
  • Active Search Indicator: Displays active filters as chips above order list
  • Filter Management: Remove individual filters or clear all at once
  • Multi-Selection: Checkboxes on ASN cards and Container cards allow selecting multiple items
  • Selection Summary: Shows count of selected SCNs and Containers
  • Dynamic Combinations: Add/Remove combination rows with Vehicle Type, Vendor, and No of Vehicles
  • Real-time Validation: Vehicle count validation against selected containers
  • Allocation Display: Shows how many containers are allocated to each combination
  • Max Limit Calculation: Each combination shows max vehicles based on remaining containers
  • Multiple TO Creation: Button shows count of TOs to be created
